Issue #2: String Concatenation
─────────────────────────────────
File: src/Evalvisitor.cpp:107-116
Problem: repeated += causes O(n²) allocations
Example: s *= 1000000 causes 500 billion char copies
Memory: >1GB temporary allocations for 1MB string
Result: Timeout + memory exhaustion

Solution: Pre-allocate with reserve()
Code: repeated.reserve(s.length() * count);
Impact: Tests 37, 47, 56, 70 will PASS
